# Usage

## Installing Homebrew

For those who don't have homebrew installed,

0. Go to official [**homebrew**](https://brew.sh/) website(click Link)

<p align="center">
    <img src="images\copy_homebrew_install_command.png" width="70%" >
<p align="center">

<p align="center">
1. Click Copy Button(Click It!)
<p align="center">

<p align="center">
    <img src="images\copy_to_terminal.png" width="70%" >
<p align="center">


2. Open **terminal**, press <kbd>Ctrl+V</kbd> ( <kbd>Cmd+V</kbd> on **Mac** ) to paste theses command, and Install Homebrew, after that, run the following commands in order.



## Installing the App
```zsh 
brew tap fabyday/kawaikara
brew install --no-quarantine --cask kawaikara
```


## Update the App
```zsh
brew upgrade --no-quarantine --cask kawaikara
```
