# 사용법

<p align="center">

<p align="center">

2. **terminal**을 키고 Ctrl+V로 복사해온 명령줄을 붙여넣어 홈브류(**homebrew**)를 설치해 줍니다. 이후에는 아래의 필요한 커맨드는 차례로 입력하세요.



## 앱 설치 
```zsh 
brew tap fabyday/kawaikara
brew install --no-quarantine --cask kawaikara
```


## 앱 업데이트
```zsh
brew upgrade --no-quarantine --cask kawaikara
```
